Projects

US Hate Crimes 2005-2011

Interactive map of hate crimes per capita within the US from 2005 to 2011. Data is based on the FBI's yearly hate crimes report and US census data. The map aims to help determine the impact state anti-discrimination legislation has on the hate crime reporting versus the hate crimes committed in each state.

Bitly Read-Time Media Map

Live, interactive data visualization I helped develop as an intern at Bitly. The map looks at how media properties are being disproportionately consumed online. For more information on how to interpret the map and how it was built read the blog post.

Tyrmt: Waterproof Mation Capture

Motion capture system for athletes and researchers. Developed as a part of my senior design project at GWU. Features both an embedded system designed for underwater motion tracking and wireless data transmission, as well as PC based software for analyzing motion efficiency.

Primacron: CNC

Home built 3-axis CNC. I constructed both the CNC and wrote the control software for the project. Control software features a PC based software for parsing G-Code as well as firmware for an Arduino microcontroller to power the motors.
